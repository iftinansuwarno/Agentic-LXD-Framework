# Curriculum Alignment & Mapping Sample
**Project:** Agentic AI Framework for Higher Education (JHU Context)

## 1. Alignment Table
| Level | Identifier | Description | Taxonomy Level |
| :--- | :--- | :--- | :--- |
| **PLO** | PLO-01 | Graduates will design AI-integrated learning environments. | Create (Level 6) |
| **CLO** | CLO-1.2 | Students will implement RAG-based agents to automate feedback. | Apply (Level 3) |
| **MLO** | MLO-1.2.1 | Configure a retrieval system using pedagogical metadata. | Understand (Level 2) |

## 2. Methodology
Mapping ini dilakukan dengan mensintesis protokol **CDC (Curriculum Development Center)** dan menyesuaikannya dengan kapabilitas **LLM (Large Language Models)**. Tujuannya adalah memastikan bahwa setiap "Agent" yang dibuat tetap berada dalam koridor kurikulum yang telah ditentukan.

## 3. Tooling
- **Mapping Logic:** Standardized via Python (see `ai-logic/` folder).
- **Compliance:** Aligned with **ADA** and **UDL** (Universal Design for Learning) standards.
