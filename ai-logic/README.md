# AI Interaction Logic & Prompting Framework
The framework mandates a **Human-in-the-Loop** architecture to prevent AI dependency.

## The Interaction Cycle
1.  **Orientation**: Students prompt AI for context and analogies (e.g., lifestyle factors in kidney failure).
2.  **Investigation**: AI explains biological mechanisms (e.g., glomerular filtration).
3.  **Validation (Critical Step)**: Mandatory cross-verification with peer-reviewed sources.
4.  **Development**: AI helps format findings into creative products (PSA/Posters).
5.  **Reflection**: Students submit a **Prompt Log** to identify potential AI hallucinations.
